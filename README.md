## Compilation

Compiler puis executer le serveur avant d'executer les clients

### Serveur 

gcc serveur.c -o exec (nom de l'executable) -lpthread
./exec


### Client 

gcc client.c -o exec (nom de l'executable) -lpthread
./exec
